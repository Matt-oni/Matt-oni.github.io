About me

I am a passionate Game Programmer with a strong background in both game development and software engineering. With a Technical Diploma in Game Development from ITS Academy "LazioDigital" (graduated with honors), I have cultivated a deep understanding of C++ programming, Unreal Engine, and game design principles. My journey into the gaming industry is driven by an ambition to create innovative and immersive experiences, combining creativity with technical expertise.

Experience

I have worked as a Game Programming Instructor at Red Hog Studio, where I taught courses on Unreal Engine Basics and C++ programming, helping aspiring developers build a strong foundation in game development. Prior to that, I completed a 720-hour internship as a Game Developer, where I contributed to an endless runner game using Unreal Engine 5, implementing core mechanics, AI, and mobile optimization.

Beyond game development, I have experience as a Full-Stack Developer and Software Engineer, working with Java, Spring Boot, Angular, Docker, and cloud technologies on enterprise applications.

Skills & Technologies

Game Development: Unreal Engine 5, C++, Blueprints, Unity, C#, Shader Programming
Software Engineering: Java, Spring Boot, SQL, Microservices, Docker, Cloud Computing
Tools & IDEs: Rider, Visual Studio Code, Git, Jenkins, Kubernetes
Game Design & AI: AI behavior scripting, procedural mechanics, game balancing

Passion for Gaming

I have always been fascinated by video games, not just as a player but as a creator. My passion lies in experimenting with new gameplay mechanics, AI-driven behaviors, and procedural systems to enhance player experiences. I thrive in collaborative environments, where I can push the boundaries of game development and continuously learn new technologies.
